# 📔 Bullet Journal

나만의 디지털 불렛저널입니다. 매일 기록하고, 할 일을 체크하고, 습관을 추적하고, 목표를 관리하세요.

## 구성

| 파일 | 설명 |
|------|------|
| [`monthly-log.md`](./monthly-log.md) | 월간 로그 — 한 달의 일정과 할 일을 한눈에 |
| [`weekly-log.md`](./weekly-log.md) | 주간 로그 — 요일별 계획과 회고 |
| [`daily-log.md`](./daily-log.md) | 일간 로그 — 매일의 할 일·이벤트·메모 |
| [`habit-tracker.md`](./habit-tracker.md) | 습관 트래커 — 매일 체크하는 습관 표 |
| [`goals.md`](./goals.md) | 목표 & 메모 — 분기/연간 목표와 모아두는 메모 |
| [`index.html`](./index.html) | 인터랙티브 웹앱 버전 (체크 토글, 진행률 표시) |

## 사용하는 기호 (Bullet Journal Key)

| 기호 | 의미 |
|------|------|
| `- [ ]` | 할 일 (Task) |
| `- [x]` | 완료 (Done) |
| `> ` | 이동/연기 (Migrated) |
| `○` | 이벤트 (Event) |
| `–` | 메모 (Note) |
| `★` | 중요 (Priority) |

## 사용법

1. 매일 `daily-log.md`에 그날의 할 일과 메모를 적습니다.
2. 한 주가 시작될 때 `weekly-log.md`를 채웁니다.
3. 매일 밤 `habit-tracker.md`에서 지킨 습관을 체크합니다.
4. 완료한 할 일은 `- [ ]`를 `- [x]`로 바꿉니다.
5. 보기 좋게 보고 싶으면 `index.html`을 브라우저로 엽니다.

## 🖥️ 여러 기기에서 쓰기 (집 ↔ 회사)

마크다운 파일은 GitHub에 저장되므로, **인터넷 브라우저 + GitHub 로그인**만 있으면
집이든 회사든 어디서나 같은 내용을 보고 편집할 수 있습니다. (앱 설치 불필요)

**방법 1 — 웹에서 바로 편집 (가장 간단)**

1. 회사/집 브라우저에서 이 저장소로 접속: `github.com/yjyj41/bullet-journal`
2. 편집할 파일(예: `daily-log.md`)을 클릭합니다.
3. 오른쪽 위 **연필(✏️) 아이콘**을 누릅니다.
4. 내용을 적은 뒤 초록색 **Commit changes** 버튼을 누릅니다. → 저장 완료!
5. 다른 컴퓨터에서 다시 열면 방금 적은 내용이 그대로 보입니다.

**방법 2 — 키보드 단축키로 편집기 열기**

- 저장소 페이지에서 키보드 **`.`(마침표)** 를 누르면 `github.dev` 웹 편집기가 열립니다.
- VS Code와 똑같은 화면에서 여러 파일을 편집하고, 왼쪽 소스 컨트롤 탭에서 커밋하면 저장됩니다.

> ⚠️ HTML 앱(`index.html`)은 입력 내용이 그 브라우저에만 저장되어 기기 간 동기화가 안 됩니다.
> 여러 기기에서 같은 기록을 보려면 위처럼 **마크다운 파일을 GitHub에서 편집**하세요.
> HTML 앱은 한 기기에서 빠르게 체크/확인하는 용도로 쓰면 좋습니다.
