# 📔 Bullet Journal

나만의 디지털 불렛저널입니다. 매일 기록하고, 할 일을 체크하고, 습관을 추적하고, 목표를 관리하세요.

## 구성

| 파일 | 설명 |
|------|------|
| [`monthly-log.md`](./monthly-log.md) | 월간 로그 — 한 달의 일정과 할 일을 한눈에 |
| [`weekly-log.md`](./weekly-log.md) | 주간 로그 — 요일별 계획과 회고 |
| [`daily-log.md`](./daily-log.md) | 일간 로그 — 매일의 할 일·이벤트·메모 |
| [`habit-tracker.md`](./habit-tracker.md) | 습관 트래커 — 매일 체크하는 습관 표 |
| [`goals.md`](./goals.md) | 목표 & 메모 — 분기/연간 목표와 모아두는 메모 |
| [`index.html`](./index.html) | 인터랙티브 웹앱 버전 (체크 토글, 진행률 표시) |

## 사용하는 기호 (Bullet Journal Key)

| 기호 | 의미 |
|------|------|
| `- [ ]` | 할 일 (Task) |
| `- [x]` | 완료 (Done) |
| `> ` | 이동/연기 (Migrated) |
| `○` | 이벤트 (Event) |
| `–` | 메모 (Note) |
| `★` | 중요 (Priority) |

## 사용법

1. 매일 `daily-log.md`에 그날의 할 일과 메모를 적습니다.
2. 한 주가 시작될 때 `weekly-log.md`를 채웁니다.
3. 매일 밤 `habit-tracker.md`에서 지킨 습관을 체크합니다.
4. 완료한 할 일은 `- [ ]`를 `- [x]`로 바꿉니다.
5. 보기 좋게 보고 싶으면 `index.html`을 브라우저로 엽니다.
